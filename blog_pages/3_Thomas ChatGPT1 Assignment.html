Discussion

When initially using ChatGPT prior to this assignment, I was impressed with the improvements between it and many other prior chatbots, but I was not surprised by the results of the general questions for this assignment. However, I was quite impressed with the handling of the programming queries. I was not expecting the model to be able to generate code that was instantly usable or only required minor debugging.
